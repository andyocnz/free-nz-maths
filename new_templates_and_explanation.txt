Here is an explanation of how to add new question templates, along with the full JSON for new templates you can add for the skills that currently only have one.

## How to Add New Templates

To add more questions for a skill, you need to add a new **template object** to the `templates` array for that skill in the `Curriculum.json` file. Each template object has the following structure:

*   **`id`**: A unique string identifier for the template (e.g., "Y6.M.TIME.T2").
*   **`stem`**: The question text. Use placeholders like `{paramName}` for random values that will be generated.
*   **`params`**: An object where each key is a `paramName` from the `stem`, and its value is an array defining the parameter's type and range.
    *   `["int", min, max]` for an integer between `min` and `max` (inclusive).
    *   `["decimal", min, max, decimalPlaces]` for a decimal number.
    *   `["time"]` for a random time in HH:MM format.
    *   `["choice", val1, val2, ...]` to pick a random value from the list.
*   **`answer`**: A JavaScript expression or function call that calculates the correct answer using the generated parameter names. You can use helper functions from `mathHelpers.js`.

### Year 6

**Skill: `Y6.M.TIME` - Time, schedules and elapsed time**
(Add these to the `templates` array of the skill with `id: "Y6.M.TIME"`)

```json
{
  "id": "Y6.M.TIME.T2",
  "stem": "How many seconds are in {minutes} minutes?",
  "params": { "minutes": ["int", 2, 60] },
  "answer": "minutes * 60"
},
{
  "id": "Y6.M.TIME.T3",
  "stem": "Convert {hours} hours to minutes.",
  "params": { "hours": ["int", 1, 12] },
  "answer": "hours * 60"
}
```
